# public、private、static 和 void
public、private、static 和 void 是修饰符（Modifiers）和返回类型，它们用来控制代码的访问权限、生命周期和行为。

我们可以把它们分为三类来理解：

## 1. 访问控制修饰符 (Access Modifiers)
决定“谁”可以使用这个变量或方法。

### public (公开的)
**_含义_**：完全公开。  
**_范围_**：任何地方都可以访问（同一个类、同一个包、不同包的子类、不同包的非子类）。  
**_场景_**：通常用于主方法 main、对外提供的接口方法、或者需要被其他类调用的工具方法。  
**_例子_**：public static void main(...) 必须是 public，因为 JVM（Java 虚拟机）需要从外部调用它来启动程序。

### private (私有的)
**_含义_**：严格保密。  
**_范围_**：只有当前类内部可以访问。其他类（哪怕是子类）都看不见。  
**_场景_**：用于隐藏类的内部实现细节（封装性）。比如一个用户的“密码”字段，应该设为 private，防止外部直接修改。  
**_例子_**：private String password = "123456"; // 外部无法直接读取或修改      
**_补充_**：还有两个没问到的：protected (同包或子类可见) 和 default (不写关键字，仅同包可见)。

## 2. 静态修饰符 (Static Modifier)

### static (静态的)
**_含义_**：属于类，而不属于对象。  
**_特点_**： 
- 无需实例化：不需要 new 一个对象就可以直接使用。
- 共享内存：所有该类的对象共享同一个 static 变量。

**_限制_**：static 方法中不能直接访问非 static 的变量或方法（因为非 static 变量必须有了对象才存在，而 static 方法运行时可能还没有对象）。  
**_场景_**：
* main 方法：程序入口，JVM 还没创建对象，所以必须是 static。
* 工具方法：如 Math.sqrt()，不需要 new Math 对象就能用。  
* 常量：如 static final int MAX_COUNT = 100;。

**_例子_**：
```
    class Counter {
        static int count = 0; // 所有对象共享这个计数
    }
    // 调用：Counter.count = 5; (不需要 new Counter())
```
## 3. 返回类型 (Return Type)

### void (空的/无返回值)
**_含义_**：这个方法不返回任何结果。  
**_位置_**：放在方法名之前，表示方法的“产出”。  
**_场景_**：当方法只负责“做事情”（如打印输出、修改数据、发送请求），而不需要告诉调用者“结果是什么”时。  
**_例子_**：
```
     public void printHello() {
        System.out.println("Hello"); 
        // 这里不能有 return 123; 只能有 return; (直接结束) 或者什么都不写
    }
``` 
**_对比_**：如果是 public int add(int a, int b)，则必须 return a + b;。

# 综合示例：拆解 public static void main(String[] args)

这是 Java 程序的入口，每个词都有严格的意义：
```
public static void main(String[] args) {
    // ...
}
```
**_public_**: JVM 需要从外部调用这个方法，所以必须公开。  
**_static_**: JVM 在启动时还没有创建任何对象，所以这个方法必须属于类本身，直接通过类名调用。  
**_void_**: 程序运行结束后，main 方法不需要给 JVM 返回一个具体的数值（如果要退出状态码，是用 System.exit(0)）。  
**_main_**: 方法的名字，JVM 规定死的入口名称。  
**_String[] args_**: 接收命令行参数的数组。

## 一张表总结  

|关键字   |类别   |核心含义   |通俗比喻|  
|------|-----|---------|------|  
|public   |访问控制   |谁都能用   |广场上的大喇叭，所有人都能听到。|  
|private  |访问控制   |只有自己能看   |日记本，锁在抽屉里，只有自己能看到。|  
|static   |成员属性   |属于类，共享，无需 new   |公司的总机号码，所有人都打这个号，不需要给每个员工配一个总机。|  
|void   |返回类型   |没返回值   |去倒垃圾，任务做完了就结束了，不需要带回来什么东西。|
