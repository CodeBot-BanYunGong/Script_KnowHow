`vi` (或者增强版 `vim`) 是 Linux 系统中最经典、最通用的文本编辑器。对于初学者来说，最困惑的是它的**“模式”**概念。

只要掌握了以下核心逻辑和常用命令，你就能自如地使用它。

---

### 核心概念：三种模式

`vi` 不像记事本那样打开就能打字，它有三种状态，**一定要分清当前在哪个模式**：

1.  **一般模式 (Normal Mode)**：
    *   **默认模式**。一打开文件就是这个模式。
    *   在这个模式下，按键被视为**命令**（如复制、粘贴、删除、移动光标），**不能直接打字**。
    *   *任何时候按 `Esc` 键，都会回到这个模式。*

2.  **编辑模式 (Insert Mode)**：
    *   在这个模式下，你才可以像在 Windows 记事本里一样**输入文字**。
    *   屏幕左下角通常会显示 `-- INSERT --` 或 `-- 插入 --`。

3.  **底行命令模式 (Command-line Mode)**：
    *   用于保存、退出、搜索等。
    *   在一般模式下输入 `:` (冒号) 进入，光标会跳到屏幕最下方。

---

### 1. 打开与退出 (最基本的一套)

| 操作 | 命令 | 说明 |
| :--- | :--- | :--- |
| **打开文件** | `vi filename` | 打开或新建一个文件 |
| **进入编辑** | 按 `i` | 从一般模式进入编辑模式 (Insert)，开始写字 |
| **回到一般模式** | 按 `Esc` | **写完字想保存，必须先按 Esc 退出编辑模式** |
| **保存** | `:w` | Write，写入磁盘 |
| **退出** | `:q` | Quit，退出编辑器 |
| **保存并退出** | `:wq` | 最常用的组合命令 |
| **强制退出** | `:q!` | **不保存**直接退出 (改乱了想重来时用) |

---

### 2. 切换到编辑模式 (如何开始打字)

在**一般模式**下，按以下键进入编辑模式：

*   **`i`**：在当前光标**前**开始插入 (Insert) -> **最常用**。
*   `a`：在当前光标**后**开始插入 (Append)。
*   `o`：在当前行**下方**新开一行开始插入 (Open)。
*   `O` (大写)：在当前行**上方**新开一行开始插入。

---

### 3. 光标移动 (在一般模式下)

虽然方向键 `↑` `↓` `←` `→` 通常也能用，但 `vi` 的核心键位是：

*   **`h`** (左)   **`j`** (下)   **`k`** (上)   **`l`** (右)
*   **`gg`**：跳到文件**第一行**。
*   **`G`** (大写)：跳到文件**最后一行**。
*   **`^`** 或 **`0`**：跳到本行**行首**。
*   **`$`**：跳到本行**行尾**。
*   `:n`：跳到第 n 行 (例如 `:10` 跳到第 10 行)。

---

### 4. 删除、复制与粘贴 (在一般模式下)

`vi` 的删除和复制非常高效，不需要鼠标选中：

*   **删除 (其实是剪切)**：
    *   **`x`**：删除光标所在的**一个字符**。
    *   **`dd`**：删除光标所在的**整行** (最常用)。
    *   `ndd`：删除向下 n 行 (例如 `5dd` 删除 5 行)。
    *   `dG`：删除从光标处到文件末尾的所有内容。

*   **复制**：
    *   **`yy`**：复制光标所在的**整行**。
    *   `nyy`：复制向下 n 行 (例如 `5yy` 复制 5 行)。

*   **粘贴**：
    *   **`p`** (小写)：在光标**下一行**粘贴。
    *   `P` (大写)：在光标**上一行**粘贴。

---

### 5. 撤销与重做 (后悔药)

如果在一般模式下误删了东西：

*   **`u`**：**撤销**上一步操作 (Undo)。
*   **`Ctrl + r`**：**重做** (撤销刚才的撤销)。

---

### 6. 查找与替换

在一般模式下：

*   **查找**：
    *   输入 **`/keyword`** 然后回车：向下查找 "keyword"。
    *   按 **`n`**：查找下一个。
    *   按 **`N`** (大写)：查找上一个。

*   **替换** (底行模式)：
    *   `:s/old/new`：替换当前行第一个匹配的词。
    *   `:s/old/new/g`：替换当前行所有匹配的词。
    *   **`:%s/old/new/g`**：替换**整个文件**中所有的 "old" 为 "new" (非常有用)。

---

### 救急指南 (如果卡住了)

如果你不知道自己按了什么，界面乱了或者动不了：

1.  狂按几次 **`Esc`** (确保回到一般模式)。
2.  输入 **`:q!`** 然后回车 (不保存强制退出)。
3.  重新打开文件再来一次。
